# Expense Tracker

A modern, production‑ready expense‑tracker application built with:
- **Frontend**: React 19, Vite, TypeScript, Tailwind CSS, Redux Toolkit, React Router, Axios, Recharts
- **Backend**: Node .js, Express, TypeScript, Prisma ORM, PostgreSQL, JWT authentication
- **DevOps**: Docker, Docker‑Compose, Kubernetes manifests, GitHub Actions CI/CD

The repository follows a monorepo layout with separate `frontend` and `backend` packages.
